# Violet

Instagram Analytics app that tracks a users post, likes, comments, shares, saves, followers, and following. Making the data easily digestible for the average user.

Requirements:
cd backend
pip install flask
pip install flask-cors


To run website:
cd backend
python app.py
(ctrl+c to close)

cd frontend
npm run dev
click on http://localhost:5173/
(ctrl+c to close)

On website:
Enter instagram login. Make sure authentication app 2FA is enabled. 
SMS 2FA is not supported and will not work.
